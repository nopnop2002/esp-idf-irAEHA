# How to build

```
git clone https://github.com/nopnop2002/esp-idf-irAEHA
cd esp-idf-irAEHA/esp-idf-irAEHA-StickC
make menuconfig
make flash
```

\*このアプリ特有のメニュー項目はありません。

--- 

# How to use

ボタンB(サイドボタン)を押してIRコードを選択します。
ボタンBを2秒以上押すと、初期画面が表示されます。
ButtonA(Front Button)を押してIRコードを発射します。
