# How to build

```
git clone https://github.com/nopnop2002/esp-idf-irAEHA
cd esp-idf-irAEHA/esp-idf-irAEHA-Stick
make menuconfig
make flash
```

\*このアプリ特有のメニュー項目はありません。

--- 

# How to use

ボタンを押してIRコードを選択します。
ボタンを2秒以上押すと、IRコードを発行します。
